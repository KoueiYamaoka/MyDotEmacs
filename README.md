# MyDotEmacs

## How to Install
### install mercurial for git
```sh
$ sudo apt-get insatll mercurial
```

### for python
```sh
$ sudo pip install yapf
$ sudo pip install flake8
$ sudo pip install virtualenv
```

### install aspell
```sh
sudo apt-get install aspell aspell-en
```

then, run emacs

## alias
```sh
e='emacsclient -nw -a ""'
ekill='emacsclient -e "(kill-emacs)"'
```
